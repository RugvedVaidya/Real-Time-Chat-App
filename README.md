# Real-Time Chat Backend

A scalable real-time chat backend built using Node.js, Express.js, Socket.IO, MongoDB, Redis, and JWT authentication.

## Features Implemented
- User Registration
- Login Authentication
- JWT Access Tokens
- Refresh Token Flow
- Logout System
- Protected Routes
- Socket.IO Setup

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Socket.IO
- JWT
- bcrypt
- Redis
- BullMQ (Upcoming)
